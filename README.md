# 🤖 AI Resume Screening System

An intelligent system that ranks resumes based on job description using NLP.

## 🚀 Features
- Upload multiple resumes (PDF)
- Paste job description
- Automatic resume ranking
- Download results

## 🛠 Tech Stack
- Python
- Streamlit
- Scikit-learn
- PDFPlumber

## ▶️ How to Run
pip install -r requirements.txt  
streamlit run app.py

## 📊 Output
The system ranks resumes based on similarity score with job description.
